1️⃣ Project Title
# E-Commerce Website Manual Testing Project

2️⃣ Project Description
This project involves manual testing of a demo E-Commerce website.
The goal was to design, execute, and document test cases covering
core modules such as Login, Registration, Cart, Checkout, and Payment.

3️⃣ Website Tested
Website Name: Automation Exercise  
URL: https://automationexercise.com

4️⃣ Modules Covered
- User Registration
- Login
- Product Page
- Add to Cart
- Cart Management
- Checkout
- Payment
- UI & Navigation
  
5️⃣ Testing Types Performed

- Regression Testing
- UI Testing
- Compatibility Testing
- Negative Testing
- Basic Performance Testing
